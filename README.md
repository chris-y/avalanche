# Avalanche

Avalanche is a lightweight ReAction GUI for xadmaster, xfdmaster and Deark, designed for AmigaOS 3.2 and 4.1. It offers a tabbed, modern interface for browsing, extracting, creating and editing archives, with optional virus scanning and built‑in disk image support.

**Features**

- Tabbed interface – open multiple archives in one window  
- Open, browse and extract archives via xadmaster.library and xfdmaster.library  
- Create and edit LhA archives (requires LhA command‑line tool)  
- Create and edit Zip archives – OS3.2 uses Zip 2.32, OS4 uses zip.library
- Optional virus scanning after extraction (xvs.library)  
- Disk image support – open ADF/DMS files directly in Avalanche; on OS3.2, double‑click ADFs to extract and mount on Workbench  
- Drag‑and‑drop for opening archives and adding files  
- Browser view with sorting and directory navigation  
- Multiple simultaneous windows 
- ARexx port for scripting (sorry, quite minimal)
- Commodity registration with hotkey activation  
- Localisation via AmigaOS catalogs  
- ToolType and GUI preference configuration
